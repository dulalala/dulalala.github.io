hexo+github搭建个人博客

hexo是目前最主流的静态博客框架，该框架基于node，因此编译效率非常高，同事借助github可以实现零成本搭建和维护，Let’s go！

环境搭建

安装node环境

在node官网下载对应操作系统的node安装包，安装过程比较简单，就是next模式，安装完毕后通过以下步骤检测是否安装成功：

  • win+r打开命令行窗口
  • 输入cmd回车
  • 在cmd窗口输入node -v

如果提示“node不是外部或内部命令”，则表示安装失败
git：版本控制器

安装git

进入git官网下载对应操作系统的git安装包，next模式安装

安装vscode编译器

vscode是一款强大的IDE

安装hexo脚手架

  1. win+r输入cmd回车

  2. 输入如下命令，回车

    • npm：依赖包管理器
    • install：安装、导入，可以简写为i
    • -g：全局安装

清理缓存

当修改了博客内容，编译后浏览器页面没有发生变化，需要清理博客缓存，方式如下：

  1. 在blog目录执行hexo clean后重新编译
  2. 在浏览器博客页面使用ctrl+shift+i调出控制台，切换到network，选中disable cache，然后ctrl+r(刷新)

替换主题

  1. 在主题官网找到自己喜欢的主题，根据提示git到本地themes目录，
  2. 修改博客项目根路径下的_config.yml配置文件，将theme改为新主题文件夹名称
  3. 停止hexo服务再重启(ctrl+c)
  4. 执行hexo g编译，然后去浏览器预览

托管到github

  1. 注册github账号(自行百度)

  2. 验证ssk key

    • 在桌面右键git bash here
    • ssh-keygen -t rsa-C “注册git的邮箱” 命令，然后一路回车，直到出现泡沫团
    • 进入C:\users\计算机用户名.ssh文件夹
    • 右键打开id_rsa.pub文件，然后ctrl+a全选，然后ctrl+c复制
    • 进入github官网，右上角点击个人头像，下拉菜单选择settings
    • 选择左侧ssh and gpg keyd
    • 单击右侧add ssh key按钮
    • title：随便填入，key：ctrl+v，然后点击add ssh key按钮

  3. 验证github用户名和邮箱

    • 在桌面右键git bash here，依次执行以下命令

  4. 在github创建用户名同名仓库

    • 点击github头像左侧+按钮，选择第一项
    • 仓库名: github用户名.github.io
      ps:注意区分大小写，必须保持一致

  5. 配置_config.yml

    • 打开blog项目根目录的_config.yml，找到最下面的deploy选项，按照如下格式填写：
      deploy:
  type:git
  repository:http://github.com/用户名/用户名.github.io
  branch:master
      ps：注意缩进和符号以及属性前面的空格

  6. 在blog根目录执行hexo d命令将本地博客代码推送到github服务器
    ps：不需要每更改一次都推送，先在本地测试准确无误后再推送

个人域名绑定

1. 域名申请(自行百度)
2. 域名解析，添加如下记录值
记录类型 主机记录 解析线路(isp) 记录值
CNAME www 默认 用户名.github.io
A @ 默认 192.30.252.153
A @ 默认 192.30.252.154 
3. 在博客主目录下找到public文件夹
    在里面新建名为CNAME的文件(不带后缀)
4. 在CNAME文件里面填写个人域名，如baidu.com
    ps：不带http、www等修饰
5. 执行hexo d命令推送

hexo+node+git 搭建个人博客

一、环境 工具的安装

1. node

一款牛叉的web服务器，目前很多大型项目都使用node搭建

- 特点：解析效率高，没有兼容性

比较适合io密集型的项目

- 版本：建议版本是6.10

- 安装：

下载相应操作系统的安装包，点击下载

如何测试？

  1. win+r 输入 cmd 回车
  2. 在新打开的命令行输入 node -v
  3. 如果能显示node版本则安装成功，否则重新安装

    2. vscode ：多语言开发工具

    3. git 版本控制器

二、安装hexo脚手架

  1. win+r 输入 cmd 回车
  2. 在cmd窗口输入 npm install -g hexo-cli
    npm：依赖包管理工具
    install：安装/导入
    -g：全局安装

    三、 初始化一个博客项目


    hexo init 项目名称（英文）
    init：初始化

    四、进入项目，添加依赖包


    1、进入到项目主目录文件夹
    2、鼠标右键选择git bash here
    3、使用npm install 命令添加依赖包
    目的：将项目所需依赖包添加到本地

    五、启动服务器


    1、在博客目录右键git bash here
    2、输入hexo server
    3、在浏览器地址栏输入：localhost：4000 回车预览
    先node 再git

    六、发表文章


    hexo new 文章名

    七、博客预览


    1、ctrl+s 保存
    2、hexo g 编译
    3、回到浏览器ctrl+r刷新
