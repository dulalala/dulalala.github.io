hexo+node+git 搭建个人博客

一、环境 工具的安装

1. node

一款牛叉的web服务器，目前很多大型项目都使用node搭建

- 特点：解析效率高，没有兼容性

比较适合io密集型的项目

- 版本：建议版本是6.10

- 安装：

下载相应操作系统的安装包，点击下载

如何测试？

  1. win+r 输入 cmd 回车
  2. 在新打开的命令行输入 node -v
  3. 如果能显示node版本则安装成功，否则重新安装

    2. vscode ：多语言开发工具

    3. git 版本控制器

二、安装hexo脚手架

  1. win+r 输入 cmd 回车
  2. 在cmd窗口输入 npm install -g hexo-cli
    npm：依赖包管理工具
    install：安装/导入
    -g：全局安装

    三、 初始化一个博客项目


    hexo init 项目名称（英文）
    init：初始化

    四、进入项目，添加依赖包


    1、进入到项目主目录文件夹
    2、鼠标右键选择git bash here
    3、使用npm install 命令添加依赖包
    目的：将项目所需依赖包添加到本地

    五、启动服务器


    1、在博客目录右键git bash here
    2、输入hexo server
    3、在浏览器地址栏输入：localhost：4000 回车预览
    先node 再git

    六、发表文章


    hexo new 文章名

    七、博客预览


    1、ctrl+s 保存
    2、hexo g 编译
    3、回到浏览器ctrl+r刷新
